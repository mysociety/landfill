API

We intend to have a full web service API. Using GET/POST requests, you should be able to submit/receive informaton which interacts the same way our web pages do. All data is available via XML for reuse.

Get posts out

All post data is available via xml. We also have a KML file for use with Google Earth.

Submit data

Fake the form effectively. Currently output will be a HTTP redirect. You get sent to /error/?MESSAGE_GOES_HERE for errors.

Search

Have a target=xml flag to search box.

Search within

Make a GET request to /cgi-bin/kml.cgi with parameters of lat/long of the top left and bottom right corners. Parameters are named: topleft_lat, topleft_long, topright_lat, topright_long. All co-ordinates are WGS84.
