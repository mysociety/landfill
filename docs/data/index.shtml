API

Get posts out

All post data is available via xml, with an RSS feed. We also have a KML file for use with Google Earth, although this currently doesn't work.

Search within

Make a GET request to /cgi-bin/kml.cgi with parameters of lat/long of the top left and bottom right corners. Parameters are named: topleft_lat, topleft_long, topright_lat, topright_long.
