All data created on Placeopedia and YourHistoryHere is being made freely available under the Creative Commons Attribution-ShareAlike license version 2.5 for use by whoever is interested. This is part of our effort to offer coders and users an easy, standard system for mapping annotation that interoperates easily with current and future geographic-based tools and systems.

Google Earth auto-update

To add Placeopedia to Google Earth, such that it automatically shows you the latest additions on the map you're looking at, auto-updating as you move around, please use this KML file.

Feeds of 50 latest additions

  • XML data
  • RSS feed
  • KML file (for Google Earth)

To restrict the RSS feed to a particular geographic area, specify a bounding box in the URL as follows:
http://www.placeopedia.com/cgi-bin/rss.cgi?topleft_long=-1;topleft_lat=54;bottomright_long=1;bottomright_lat=50
