API

We intend to have a full web service API. Using GET/POST requests, you should be able to submit/receive informaton which interacts the same way our web pages do. All data is available via XML for reuse.

Get posts out

All post data is available via xml. We also have a KML file for use with Google Earth. Both these only show the most recent 25 additions.

Submit data

Fake the form effectively. Currently output will be a HTTP redirect. You get sent to /error/?MESSAGE_GOES_HERE for errors.

Search

Have a target=xml flag to search box.

Search within

Make a GET request to /cgi-bin/kml.cgi with a BBOX parameter of long/lat of the bottom left and top right corners, separated by commas, e.g. BBOX=-4.7239,50.1146,4.18819,54.4935. All co-ordinates are WGS84.
