API

Get posts out

All post data is available via xml, with an RSS feed. We also have a KML file for use with Google Earth, although this currently doesn't work.

Search within

Make a GET request to /cgi-bin/kml.cgi with a BBOX parameter of long/lat of the bottom left and top right corners, separated by commas, e.g. BBOX=-4.7239,50.1146,4.18819,54.4935.
